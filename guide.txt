ФБУЗ «ЦГИЭ в Мурманской области» — сайт (прототип)
====================================================

КАК ПОСМОТРЕТЬ
--------------
Распакуйте папку и откройте файл index.html двойным щелчком — сайт
откроется в браузере. Для локального теста удобнее запустить простой
сервер (по желанию): в папке сайта выполните
    python -m http.server 8000
и откройте http://localhost:8000


ИЗ ЧЕГО СОСТОИТ САЙТ
--------------------
index.html        — Главная
uslugi.html       — Услуги и цены
dokumenty.html    — Документы и формы
novosti.html      — Новости (лента из ВКонтакте)
kontakty.html     — Контакты и карта
assets/
  content.js      — ВЕСЬ ТЕКСТ САЙТА (его и редактируете)
  styles.css      — оформление (НЕ трогать)
  app.js          — логика (НЕ трогать)
  logo.jpg        — логотип
files/            — сюда кладёте PDF/DOC документы (создайте папку)


ЧТО МЕНЯТЬ — ТОЛЬКО ОДИН ФАЙЛ: assets/content.js
------------------------------------------------
Откройте его в «Блокноте» или любом текстовом редакторе. Внутри —
понятные блоки с подписями: телефоны, услуги, цены, документы, новости.
Меняйте только текст в кавычках. После сохранения обновите страницу (F5).

• Добавить новость / услугу / документ — скопируйте соседний блок { ... }
  целиком (вместе с запятой) и впишите свой текст.
• Поменять телефон/адрес — раздел config в начале файла. Меняется сразу
  на всех страницах.


КАК ВЫЛОЖИТЬ ДОКУМЕНТЫ ДЛЯ СКАЧИВАНИЯ
-------------------------------------
1. Создайте на хостинге папку files.
2. Загрузите туда файл, например zayavlenie.pdf
3. В content.js в разделе documents впишите имя файла:
       file: 'files/zayavlenie.pdf'
   (сейчас там стоит '#', т.е. заглушка)


НОВОСТИ ИЗ ВКОНТАКТЕ (АВТОМАТИЧЕСКИ)
------------------------------------
Чтобы новости подтягивались из вашего сообщества сами:
1. Зайдите на vk.com/dev → «Создать приложение» → тип «Сайт».
   Укажите адрес сайта (например, fguzmo.ru). Это бесплатно, 2 минуты.
2. Скопируйте «ID приложения».
3. В content.js → config → vkAppId впишите этот ID в кавычках.
После этого на странице «Новости» появится живая лента из сообщества,
и вести новости в двух местах больше не нужно — публикуете в ВК как обычно.

Пока поле vkAppId пустое — показывается ручной список из content.js.


ВЕРСИЯ ДЛЯ СЛАБОВИДЯЩИХ
-----------------------
Уже встроена. Кнопка в верхней строке сайта. Переключает крупный шрифт
и контрастные цветовые схемы — обязательный элемент для госучреждения.


ПЕРЕД ПУБЛИКАЦИЕЙ (важное)
--------------------------
• Замените цены «уточняется» на реальные.
• Загрузите реальные документы и пропишите их в content.js.
• Наполните страницу «Политика конфиденциальности» (152-ФЗ): нужна, если
  на сайте появится форма сбора персональных данных.
• Проверьте реквизиты и сведения о лицензии на странице контактов.
