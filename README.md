# Сайт ФБУЗ «ЦГИЭ в Мурманской области»

Статический сайт. Размещается на GitHub Pages.

## Как обновлять контент (без программирования)
Весь изменяемый текст — в файле **assets/content.js**:
телефоны, услуги, цены, документы, новости, руководство, филиалы.

Редактировать можно прямо в браузере (в т.ч. с телефона):
1. Откройте файл `assets/content.js` в этом репозитории.
2. Нажмите карандаш (Edit / Изменить).
3. Поменяйте текст внутри кавычек.
4. Внизу нажмите **Commit changes**.
Через ~1 минуту сайт по адресу GitHub Pages обновится автоматически.

Документы (PDF/DOC) загружаются в папку **files/**, затем имя файла
прописывается в `assets/content.js` в разделе documents.

Файлы `assets/styles.css` и `assets/app.js` — оформление и логика,
их трогать не нужно.
